函数参考 (Codroid SDK)¶

核心控制接口 (Control Interface)¶

这里是 SDK 的主要入口类，负责机器人连接、模式切换、IO 控制及运动指令发送。 This is the main entry class of the SDK, responsible for robot connection, mode switching, IO control and motion command sending.

codroid.CodroidControlInterface.CodroidControlInterface ¶

Codroid 机器人控制接口 / Codroid Robot Control Interface.

__init__(host='192.168.1.136', port=9001) ¶

初始化控制接口 / Initialize the control interface.

参数：

  • host (str, 默认： '192.168.1.136' ) –

    机器人 IP 地址 / Robot IP address.

  • port (int, 默认： 9001 ) –

    端口号 (默认 9001) / Port number (default 9001).
