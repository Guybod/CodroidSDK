数据模型与枚举 (Models & Enums)¶

本页列出了 Codroid SDK 中使用的所有标准化数据结构和枚举类型。

1. 基础响应模型 (Base Models)¶

所有的 API 请求都会返回 CodroidResponse 对象。

codroid.models.CodroidResponse dataclass ¶

SDK 通用响应对象 / Universal SDK Response Object.

属性：

  • id (Union[int, str]) –

    请求的唯一标识 ID / Unique request ID.

  • ty (str) –

    接口类型名称 / Interface type name.

  • db (Optional[Any]) –

    接口返回的具体数据内容 / Data returned by the interface.

  • err (Optional[Any]) –

    错误信息。若为 None 则表示操作成功 / Error message, None if successful.

is_success property ¶

判断请求是否成功 / Check if the request was successful.

2. 运动控制模型 (Motion Models)¶

用于配置机器人运动路径、点位和运动模式。

2.1 运动点定义 (MovePoint)¶

codroid.models.MovePoint dataclass ¶

通用运动点位定义 / General Motion Point Definition.

属性：

  • jp (Optional[Sequence[float]]) –

    关节角列表 [j1...j6] / Joint angles.

  • cp (Optional[Sequence[float]]) –

    笛卡尔坐标 [x,y,z,a,b,c] / Cartesian pose.

  • rj (Optional[Sequence[float]]) –

    逆解参考关节角 / Reference joint angles for IK.

  • ep (Optional[Sequence[float]]) –

    外部轴位置 / External axis positions.

to_dict() ¶

过滤空字段，防止后端崩溃 / Filter None to prevent server crash.

3.1 全局变量 (GlobalVariable)¶

codroid.models.GlobalVariable dataclass ¶

全局变量数据模型 / Global Variable Data Model.

属性：

  • value (Any) –

    变量值 (支持 int, float, str, list, dict) / Variable value.

  • note (Optional[str]) –

    变量备注 (nm) / Variable note.

to_robot_format() ¶

将 Python 对象转换为机器人协议要求的格式化字典。 Convert Python object to formatted dict required by robot protocol.
